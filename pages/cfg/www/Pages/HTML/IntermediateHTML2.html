HTML FORMS: COLLECTING AND STORING INFORMATION ONLINE.

The <form> tag is used to collect user input like in apllication forms, registration forms and many other web interfaces used for authentication, data collection, data input and retrieval tasks. The <form> tag defines an HTML form like:

<form>
    ...stuff to do here
</form>

HTML FORM ELEMENTS

HTML Form elements are used to specify what kind of method is used to collect form information. Information or data can be collected as:

  1. inputs (<input>)
  2. textareas (<textarea>, where you might be asked to write a cover letter or a speech)
  3. a selection option from a drop-down menu (<select>, <value>)
  4. a list of options from which one can be choosen (<option>)
  5. submission or uploading your data to a database (<submit>)

The <input> Element.

The <input> element is the most important form element. It is used to specify that a specific portion of HTML page is expecting some sort of user input. Types of user inputs are of various variations (text, files, numbers, emails, addresses, videos, urls etc) whic also means that the <input> element has many variations, depending on the type user data needed. The input element has the name, type, and initial value of input specified within its tags.

<form>
    First Name: (A TEXT INPUT)
    <input type='text' name='FirstName'></input>

    Phone Number: (A NUMERIC INPUT)
    <input type='num' name='Phone Number'></input>

    Email Address: (A email INPUT)
    <input type='email' name='Email'></input>

    Your Website: (A URL INPUT)
    <input type='url' name='website'></input>

    Gender: (A URL INPUT)
    <input type='radio' name='gender'>Male</input>

    CV: (A File INPUT)
    <input type='file' name='File'></input>
</form>

The <textarea> Element

This is used to specify a mini plain text editor for collecting worded text. This is mainly used to collect staements online, enquiries, complaints, generally things that involve a story or narrative from users. The textarea element has the name, number of rows and cols specified in its opening tags.

<form>
    Tell Us a Bit about yourself: (TEXTAREA INPUT)
    <textarea name='Onlinestalking' cols='30' rows='10'></textarea>

    Enquiries: (TEXTAREA INPUT)
    <textarea name='enquiries' cols='30' rows='10'></textarea>
</form>

AUDIOS, VIDEOS

AUDIO

To play an audio object:

<audio controls>
  <source src="horse.ogg" type="audio/ogg">
  <source src="horse.mp3" type="audio/mpeg">
  <source src="forrest_gump.mp4" type="video/mp4">
  <source src="forrest_gump.ogg" type="video/ogg">
Your browser does not support the audio element.
</audio>

To play a video:

<video width="320" height="240" controls autoplay>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
  Your browser does not support the video tag.
</video>

To embed a plugin or app:

<object width="400" height="400" data="helloworld.swf">
</object>

<object data="horse.wav">
  <param name="autoplay" value="true">
</object>

Create a Context Menu

The <menu> tag defines a list/menu of commands.

The <menu> tag is used for context menus, toolbars and for listing form controls and commands. It take the attributes type =" " (popup, toolbar or context), label = "Visible name of menu item" and the event attribute onclick = " "(window.open:opens a url, window.location.reload:refreshes a page, mailto: send an email.)

    <menuitem label="Refresh" onclick="window.location.reload();"
    icon="ico_reload.png">
    </menuitem>

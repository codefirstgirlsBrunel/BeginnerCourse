DIVS

The <div> tag specifies a section of a code. It used in CSS to specify layouts for web pages. This is how it works:

    <div id = 'bodyContaier'>
            This div will contain all my body elements.

        <div id = 'header'>
            <p>This is a header div within my body div that will conatain all my header elements; header image and page heading and sub
            headings.</p>

            <div class = 'ImgDivs'>This is a div that represents characteristics for all images. It belongs to the class 'ImgDivs'
            </div>

        </div>


        <div id = ''mainContent''>
            <p>This is a second div within my body div that will conatain all the main contents of my web page. i have identified it as 'mainContent'. </p>

            <div class = 'ImgDivs'>This is a div
            that represents characteristics for
            all images. It belongs to the class
            'ImgDivs'
                <img/>
                <img/>
            </div>
        </div>


        <div id = ''footer''>
            <p>This is a third div within my body div that will conatain the
            footer of my web page. i have identified it as 'footeer'. </p>
        </div>
    </div>
